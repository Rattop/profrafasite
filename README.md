# 🌐 Portal do Aluno - Base para Sites Pessoais

Este é o repositório do **site base para os alunos do curso de informática e programação** ministrado por [Rafael de Souza](#), que serve como um ponto de partida para criarem seus próprios sites.

> 🎓 **Objetivo:** Centralizar links, playlists de aulas, recursos e exemplos de código, facilitando o acesso dos alunos ao conteúdo e dando uma estrutura pronta para que possam desenvolver suas próprias versões.

---

## 📁 O que este site contém?

- Um layout simples e funcional com **HTML e CSS puro**
- Um **menu de navegação com dropdown** para organizar as turmas
- Um **iframe incorporando uma playlist do YouTube**
- Design limpo, responsivo e com **cores personalizadas via variáveis CSS**
- Estrutura pronta para ser **copiada, adaptada e reutilizada** pelos alunos
